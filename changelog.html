История изменений

Changelog

Все заметные обновления тренажёра собраны на одной странице.

Последние обновления

    17.04.2026
  • Выбор пары стал двусторонним: можно начинать с греческого слова или с перевода; неверные пары подсвечиваются красной обводкой.
  • Перестроена верхняя часть тренажёра: выбор базы, количество слов, описание раунда и кнопка запуска теперь расположены стабильнее.
  • Таблица слов теперь раскрывается вниз по кнопке, а свернутый список показывает первые строки с мягким затемнением.
  • Добавлена таблица со всеми словами выбранного словаря; список обновляется при смене базы.
  • Добавлен словарь character с лексикой по характеристикам и характеру.
  • Добавлен словарь familyB1 с лексикой по теме семьи.
    • 10.04.2026
  • Добавлена поддержка нескольких словарей: теперь база слов выбирается из выпадающего списка в интерфейсе.
  • Реализован манифест dictionaries/index.json, через который можно подключать тематические файлы словарей без изменения кода.
    • 17.11.2025
  • Реализован счётчик неправильных ответов и финальный экран статистики (всего пар, ошибок, точность) с центрированием на мобильных.
  • Убрана форма добавления слов через интерфейс, теперь добавляем пары напрямую в words.txt.
  • Мобильная версия переработана — все пять пар помещаются на один экран, карточки стали компактнее.
  • В подвал добавлена версия платформы и дата последнего обновления, значения подставляются из JS-констант.
  • Добавлено отображение общего количества слов в базе прямо в подписи стартового экрана и выделено жирным для наглядности.
