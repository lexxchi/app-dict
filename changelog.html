История изменений

Changelog

Все заметные обновления тренажёра собраны на одной странице.

Последние обновления

    19.04.2026
  • Добавлены словари verbs-thinking&knowing&saying и verbs-willing&deciding с глаголами по темам мышления, речи, воли и решений.
  • В тренировке Выбери перевод увеличена пауза после неправильного ответа, чтобы правильный вариант дольше оставался видимым.
  • Добавлен новый тип тренировки Выбери перевод: показывается слово или перевод и четыре варианта ответа из выбранного словаря.
  • Добавлены словари family-A, family-B1, face, character-A и character-B1.
  • Обновлен манифест dictionaries/index.json: словари по семье и характеру разделены по уровням, а устаревшие тематические словари убраны из списка.
  • В developer notes добавлено правило именования веток с шаблоном, типами изменений и примерами для словарей.
    • 17.04.2026
  • Добавлен небольшой отступ между тренажёром и списком слов, чтобы секции визуально не слипались.
  • Сессионный прогресс теперь засчитывает только найденные пары, перенесён ниже поля и показывает название выбранного словаря.
  • Добавлен прогресс по текущей сессии: видно, сколько уникальных слов выбранного словаря уже попадало в тренировку до обновления страницы.
  • Заголовок списка слов и название выбранного словаря переносятся друг под друга на узких экранах.
  • Длинные слова в карточках больше не переносятся по буквам; текст автоматически уменьшается, если не помещается или занимает слишком много строк.
  • Колонки тренажёра выровнены: карточки получили стабильную высоту, а разделитель расположен строго по центру поля.
  • Финальная статистика на мобильных теперь отображается вертикальными строками на всю ширину.
  • Подсказка раунда перенесена выше прогресса и поля, а финальный экран статистики растягивается на всю ширину на мобильных.
  • Упорядочена структура проекта: словари перенесены в одну папку, стили и JavaScript разделены на тематические файлы, документация разделена на пользовательскую и developer notes.
  • Выбор пары стал двусторонним: можно начинать с греческого слова или с перевода; неверные пары подсвечиваются красной обводкой.
  • Перестроена верхняя часть тренажёра: выбор базы, количество слов, описание раунда и кнопка запуска теперь расположены стабильнее.
  • Таблица слов теперь раскрывается вниз по кнопке, а свернутый список показывает первые строки с мягким затемнением.
  • Добавлена таблица со всеми словами выбранного словаря; список обновляется при смене базы.
  • Добавлен словарь character с лексикой по характеристикам и характеру.
  • Добавлен словарь familyB1 с лексикой по теме семьи.
    • 10.04.2026
  • Добавлена поддержка нескольких словарей: теперь база слов выбирается из выпадающего списка в интерфейсе.
  • Реализован манифест dictionaries/index.json, через который можно подключать тематические файлы словарей без изменения кода.
    • 17.11.2025
  • Реализован счётчик неправильных ответов и финальный экран статистики (всего пар, ошибок, точность) с центрированием на мобильных.
  • Убрана форма добавления слов через интерфейс, теперь добавляем пары напрямую в words.txt.
  • Мобильная версия переработана — все пять пар помещаются на один экран, карточки стали компактнее.
  • В подвал добавлена версия платформы и дата последнего обновления, значения подставляются из JS-констант.
  • Добавлено отображение общего количества слов в базе прямо в подписи стартового экрана и выделено жирным для наглядности.
